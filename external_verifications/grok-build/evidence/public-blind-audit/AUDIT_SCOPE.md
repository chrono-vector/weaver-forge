# Public blind audit — scope

## In scope

- Public discoverability of the Grok Build Witness package under Weaver Forge
- Copyability of clone/checkout instructions without private owner context
- Completeness of Docker bootstrap, environment, logging, and evidence-template fields
- Classification and submission policy clarity at the documentation layer

## Out of scope

- Running `cargo`, Docker, DotSlash, `protoc`, or the `xai-grok-pager` product
- Verifying artifact SHA-256 or build success
- Security review of Grok Build itself
- Assigning Independent Witness (C-014) status
