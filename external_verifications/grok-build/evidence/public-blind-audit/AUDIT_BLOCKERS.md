1. **No annotated package tag** — Witness cannot resolve an immutable Weaver Forge package revision from a published tag alone.
2. **Non-copyable host procedure** — Missing single orchestrator with assigned variables and safety checks.
3. **Incomplete Docker contract** — Mounts, env, platform, network, and working directory not specified as one command.
4. **Protoc descriptor hygiene** — LF-normalized writable copy, SHA-256 recording, and exact `PROTOC` export not prescribed byte-for-byte.
5. **RUSTUP_HOME policy** — Must not point Witness work dirs at the expense of the digest-pinned image toolchain.
6. **Evidence rigor** — Container vs Cargo logs, exit codes, timing, and empty-target proof not mandatory-field complete.
7. **Templates and validator** — Insufficient prescriptive fields; no structural validator for submissions.
8. **C-014** — Independent Witness remains **`NOT_STARTED`**; blind AI review does not close this.
