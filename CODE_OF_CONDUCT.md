# Code of Conduct

**Weaver Forge — Daily Commit Lab** is a builder community for technical growth, accountability, and evidence-backed progress.

### Our Standard
**Respect people. Challenge claims. Build with evidence.**

### Expected Behavior
- Treat others with respect
- Ask questions in good faith
- Give technical feedback clearly and constructively
- Challenge claims without attacking people
- Share evidence when making progress claims
- Credit others for their work
- Document failures honestly
- Keep discussions relevant to learning, building, and reviewing

### Unacceptable Behavior
- Harassment, insults, threats, or personal attacks
- Discrimination
- Spam, scams, or low-effort self-promotion
- Plagiarism or fake progress claims
- Sharing private code or personal data

### Enforcement
Maintainers may issue warnings or remove members for serious violations.

**Build. Test. Commit. Receipt. Repeat.**  
**No commit. No claim. No receipt. No authority.**
