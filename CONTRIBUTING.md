# Contributing Guidelines

Thank you for contributing to Weaver Forge.

### Core Principles
- Evidence first: Every claim must be backed by receipts (commits, tests, notebooks, results).
- Build more than you talk.
- Contributor ≠ Witness — you cannot witness your own work.

### Ways to Contribute
- Daily commits and build receipts
- Work in project pods
- Code reviews and witness reproductions
- Improving templates and documentation

### Submission Requirements
- Use the provided receipt and witness templates
- Link clear evidence
- Clearly state what your work proves and does not prove

See [RECEIPT_TEMPLATE.md](RECEIPT_TEMPLATE.md) and [WITNESS_REVIEW_TEMPLATE.md](WITNESS_REVIEW_TEMPLATE.md).

**Build. Test. Commit. Receipt. Repeat.**
